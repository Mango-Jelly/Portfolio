# Images Needed

Add these images to this folder:

## Required Images:

1. **logo.png** - Your logo image for the top-left navbar
   - Recommended size: 200x50px (or any size, height will auto-adjust to 50px)
   - Should be a PNG with transparent background

2. **background.jpg** - Your background image
   - Recommended size: 1920x1080px or larger
   - Will be displayed behind all sections with a blur effect
   - Choose something that fits your theme

3. **profile.jpg** - Your profile photo
   - Recommended size: 400x400px
   - Used in the About section

4. **project1.jpg through project4.jpg** - Screenshots of your projects
   - Recommended size: 600x400px
   - Used in the Projects section

## Tips:

- Use high-quality images
- For background: choose something subtle that won't make text hard to read
- For logo: make sure it has good contrast
- Compress images for faster loading (use tools like TinyPNG)
